# Sudoku Studio

Sudoku Studio is a Next.js + React Sudoku app with:

- Difficulty selector (`Easy`, `Medium`, `Hard`)
- Practice and Challenge modes
- Keyboard navigation and quick input
- Lives system in challenge mode
- Animated background and polished UI

## Tech Stack

- Next.js 14 (App Router)
- React 18 + TypeScript
- CSS Modules + global styles
- Tailwind configured for utility support

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Deploy

This project is configured for static export + GitHub Pages.

- Output directory: `out/`
- CNAME: `sudoku.velahidalgo.com`

```bash
npm run deploy
```
